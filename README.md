# Statistically-controllable-microstructure-reconstruction-framework
The source code of statistically controllable microstructure reconstruction framework.
Run SWNN-2T3-Train.py for training; 

Run SWNN-2T3-Test.py for 3D reconstruction; 

Run SWNN-2T3-Test-Heter-Gen.py for controllable generation.

Core Dependencies: Python 3.10+, PyTorch (with CUDA support recommended), TorchVision, OpenCV (cv2), NumPy, Pillow (PIL).
